Objetivo General del OVI

Conocer y aplicar los conceptos de diseño mediante una página web presentado allí las definiciones, funciones, desarrollo de Selectores CSS y Modelo de Caja (top, right, left, bottom) ya sea con imágenes, videos, entre otros


Objetivos sel OVI

* Diseñar una página web con contenido claro, preciso y conciso para que los usuarios estén bien informados con todo lo relacionado en Selectores CSS y Modelo de Caja (top, right, left, bottom)

* diseñar diferentes estilos y vistas que sean atractivas a la vista del usuario y así consulten nuestra página web.

* Saber aplicar los selectores CSS en la página HTML


Tematicas

* Selectores CSS

* Modelo de Caja (top, right, left, bottom)
